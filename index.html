    id INT AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(255) NOT NULL,

    estrellas INT NOT NULL CHECK (estrellas >= 1 AND estrellas <= 5),
    precio_noche DECIMAL(10,2) NOT NULL,
    descripcion TEXT,
    fecha_creacion TIMESTAMP DEFAULT CURRENT_TIMESTAMP

üîß Ejemplos de Uso

Usando cURL

Obtener todos los hoteles: 
curl -X GET http://localhost/APIHOTEL/apihotel.php
Crear nuevo hotel: 
curl -X POST http://localhost/APIHOTEL/apihotel.php \
  -H "Content-Type: application/json" \
  -d '{"nombre":"Hotel Test","ciudad":"Madrid","estrellas":3,"precio_noche":80.00}'
Actualizar hotel: 
curl -X PUT http://localhost/APIHOTEL/apihotel.php \
  -H "Content-Type: application/json" \
  -d '{"id":1,"precio_noche":150.00}'
Eliminar hotel: 
curl -X DELETE http://localhost/APIHOTEL/apihotel.php \
  -H "Content-Type: application/json" \
  -d '{"id":1}'

Usando JavaScript (Fetch API)

  .then(response => response.json())
  .then(data => console.log(data));

// Crear nuevo hotel

  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    nombre: 'Hotel JavaScript',
    ciudad: 'Barcelona',
    estrellas: 4,
    precio_noche: 120.00
  })
}).then(response => response.json())
  .then(data => console.log(data));
